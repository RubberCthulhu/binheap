Binary Heap
===========

C implementation of Binary Heaps (https://en.wikipedia.org/wiki/Binary_heap).

Build and test
--------------

Build the library only:
```
cmake .
make
```
or
```
cmake -DCMAKE_BUILD_TYPE=Release .
```

Build with tests:
```
cmake -DCMAKE_BUILD_TYPE=Debug .
make
```

Run tests:
```
make test [ARGS="-V"]
```
